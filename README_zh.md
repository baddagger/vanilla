# 特性

- 原生：为 macOS 原生开发
- 简洁：受唱片机启发的 UI 设计
- 音量与低音控制
- 实时音频频谱
- 音源管理
- 支持编辑歌曲元数据及封面

# 下载

你可以从 [GitHub Releases](https://github.com/baddagger/vanilla/releases) 下载最新版本。

应用在首次启动时可能会被 Gatekeeper 拦截，你需要在 `安全性与隐私` 设置中手动允许运行。
